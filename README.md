# doc-system-governance

Codex skill for creating, governing, and maintaining AI-friendly project documentation systems with clear ownership boundaries.

## Contents

- `SKILL.md`: skill definition and workflow
- `agents/openai.yaml`: agent-facing invocation metadata
- `references/doc-system-rules.md`: governance rules for the documentation system
- `references/file-templates.md`: starter templates for core documentation files

## Purpose

This skill helps bootstrap, repair, and maintain a project documentation system centered on clear ownership boundaries across product, architecture, design, engineering, and AI workflow records.
